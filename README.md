# To Do List

A front-end React app to manage a list of tasks.

## Features

- Create tasks
- Mark tasks as complete
- Rename tasks
- Delete tasks

## Technologies

- TypeScript
- React JS
- Redux JS
- Material UI
- ES6
- ESLint

## Usage

> Clone repository.

```sh
git clone https://github.com/ednoram/todo-list-frontend.git
```

> Run the following commands.

```sh
npm install
npm start
```
